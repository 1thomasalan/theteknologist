# The Teknologist

**Technology converging to make humanity better.**

The Teknologist is a curated intelligence resource built around the **BRAIN** framework — five daily intelligence briefs covering the most important signals in the convergence of technology and human progress.

## BRAIN Framework

| Letter | Vertical | Focus |
|--------|----------|-------|
| **B** | Biotech & Longevity | Gene therapy, longevity research, metabolic health |
| **R** | Robotics | Humanoids, autonomous systems, industrial automation |
| **A** | AI | Models, agents, reasoning systems, machine intelligence |
| **I** | Infrastructure & Energy | Nuclear, renewables, grid-scale storage |
| **N** | News | Curated signal across all five verticals |

A sixth layer, **Tek-Finds**, curates consumer and prosumer products from across all five verticals.

## Tech Stack

- **React 19** + **TypeScript** — UI framework
- **Vite 7** — Build tooling
- **Tailwind CSS 4** — Styling
- **Wouter** — Client-side routing
- **Express** — Production server
